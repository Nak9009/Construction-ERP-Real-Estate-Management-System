A comprehensive, modern enterprise resource planning system designed specifically for construction companies and real estate developers to manage their operations seamlessly.

Introduction

This system provides an end-to-end management solution split into two main architectures:

  • Frontend: Built with Next.js 16 (App Router), Tailwind CSS, and shadcn/ui for a beautiful glassmorphic interface.
  • Backend: Powered by Laravel 13, providing robust RESTful APIs, database management, and authentication services.

Prerequisites

Before installing the application, ensure your environment meets the following requirements:

Backend Server

  • PHP >= 8.2
  • Composer 2.x
  • MySQL 8.0+ or PostgreSQL
  • Laravel Herd / Valet (Optional but recommended for macOS)

Frontend Client

  • Node.js >= 22.9.0 (Crucial for Next.js 16)
  • npm >= 10.x
  • NVM (Node Version Manager) recommended

1. Backend Installation (Laravel)

1. Navigate to the backend directory and install PHP dependencies:

cd backend
composer install

2. Configure your environment variables:

cp .env.example .env
php artisan key:generate

Edit the .env file to set your DB_DATABASE, DB_USERNAME, and DB_PASSWORD.

3. Run database migrations to set up the schema:

php artisan migrate --seed

2. Frontend Installation (Next.js)

Important: Node Version

You MUST use Node.js version 22 or higher. The build will fail on Node 20.8.0 due to Next.js 16 requirements.

1. Ensure you are using the correct Node version:

nvm install 22
nvm use 22

2. Navigate to the frontend directory and install dependencies:

cd frontend
npm install

3. Configure the frontend environment variables (if any) and verify the build:

npm run build

Starting the Application

To run the system locally, you need to run both the backend API and the frontend development server simultaneously.

Start Backend

In a new terminal tab:

cd backend
php artisan serve

Usually runs on http://localhost:8000

Start Frontend

In another terminal tab:

cd frontend
nvm use 22
npm run dev

Usually runs on http://localhost:3000

System Navigation

Once you open http://localhost:3000, you will be greeted by the Dashboard. The application is navigated primarily via the left-hand Sidebar.

The sidebar is highly organized with collapsible sections to keep the interface clean:

  • Main: Core views like Dashboard and Projects.
  • Real Estate: Inventory management (Houses, Land), Sales, and Customers.
  • Construction: On-site progress, Warranty/Maintenance, and Documents.
  • Logistics: Procurement, Warehouse (Inventory), and Equipment tracking.
  • Finance: Financial overviews and reporting.
  • Settings: Company profile, Users, and Roles management.

Core Modules Overview

Inventory (Land & Houses)

Manage plots of land, track housing developments, monitor property statuses (Available, Sold, Reserved), and track pricing details.

Sales & Financials

Record transactions, monitor cash flow, generate invoices for property sales, and get a high-level view of company revenue versus expenses.

Procurement & Warehouse

Generate purchase orders (POs) for raw materials (cement, steel, etc.) and track stock levels in the warehouse. Get alerts when supplies run low.

Workforce (Employees & Contractors)

Manage human resources. Track internal employees, their roles and departments, as well as external subcontractors and their active contracts.
